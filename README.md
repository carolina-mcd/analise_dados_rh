# 📊 Análise de Dados — RH / Pessoas

Projeto de estudo de análise de dados com foco em Recursos Humanos, desenvolvido com Python no Google Colab.

## 🎯 Objetivo
Explorar um dataset fictício de 300 funcionários para extrair insights sobre turnover, desempenho, salários e perfil dos colaboradores.

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📁 Estrutura do projeto
- `rh_dataset.csv` — Base de dados com 300 funcionários
- `analise_rh.py` — Script completo de análise

## 💡 Principais descobertas
- Comercial tem a maior média de horas de treinamento (~67h/ano)
- Não há relação clara entre promoções e desempenho
- O fator que mais influencia o turnover é o tempo de empresa (36%)
- Taxa geral de turnover: 21%

## 🤖 Modelo de Machine Learning
Random Forest para prever risco de turnover com 80% de acurácia.
